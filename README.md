# RAG with Ollama

A simple RAG (Retrieval-Augmented Generation) system using Ollama and ChromaDB.

## Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Make sure Ollama is running
ollama serve
```

## Usage

1. Add documents to `data/documents/`
2. Run the app:

```bash
python src/index.py
```
